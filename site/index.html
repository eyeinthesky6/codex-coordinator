It does not forget the open loops

Work keeps moving after the first update.

Coordinator remembers what is still open, checks the right Codex tasks on a quiet schedule, and comes back when work finishes, gets stuck, or needs you.

01

Quiet scheduled follow-up

While a job is unfinished, Coordinator checks only the Codex tasks that changed and any handoffs still waiting. If nothing important changed, it stays quiet.

02

The right message reaches the right task

Every part of the job stays tied to its exact Codex task. Outdated instructions from an older owner or earlier work cycle are ignored.

03

Nothing disappears between windows

Finished work, queued work, blockers, and decisions waiting for you return to the Coordinator task as one project update.

Optional Mission Control

See what is moving without opening every task.

Mission Control is a local dashboard for the days when several Codex tasks are running. It shows what is active, what needs you, and when 2 tasks may be touching the same files.

Know where to lookOne screen shows which tasks are working, waiting, or finished.
Catch crossed wiresIt warns when active tasks may be changing the same files.
Keep it privateIt runs on your computer. There is no Mission Control account or cloud service.

Completely optional. Coordinator works without the dashboard.

Doctor

Two levels of review.
Only one uses AI.

Run now or on your schedule

Regular Doctor

Checks the installed Coordinator, repairs trusted files when needed, and looks for concrete problems in unfinished task handoffs.

  • Zero AI model calls
  • Does not read your application code or chat messages
  • Can run through a user-approved schedule
Optional · you click it

AI Deep Review

Asks your configured Codex model whether a task looks too small for its own window or no longer matches the job it received.

  • Low reasoning with a small, redacted task summary
  • Shows its token use
  • Suggests only—it cannot change or repair work

Small by design

Zero third-party runtime dependencies

The shipped plugin uses Python's standard library. There is no orchestration framework, database, queue, pip package, or npm package to install.

Community install today

Installable from this GitHub marketplace

The repository ships a Codex marketplace package. It is not currently listed in OpenAI’s public Plugin Directory.
