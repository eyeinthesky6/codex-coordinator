# Support

Support is handled through GitHub issues once the repository is hosted.

- **Something is broken:** use the **Bug report** form and include a minimal reproduction with secrets removed.
- **You need help using the plugin:** use the **Usage question** form.
- **You want a behavior change:** use the **Feature request** form.
- **You found a security issue:** follow [SECURITY.md](SECURITY.md), not a public issue.

Please do not paste private task messages, credentials, or a project's live coordination state into an issue.
